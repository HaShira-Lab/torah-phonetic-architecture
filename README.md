Torah Phonetic Architecture

Multi-layer computational analysis of phonetic structure in the Pentateuch.

Reproducible code and protocols for:
- Layer A (consonantal structure)
- Layer B (phonetic chains and boundaries)
- Layer C (syllabic / rhyme structure)
