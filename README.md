# Torah Phonetic Architecture

Multi-layer phonetic and poetic analysis of the Torah.

## Overview
This project investigates the Torah as a structured phonetic composition using reproducible computational methods.

We analyze three layers:

- Layer A — Consonant recurrence structure
- Layer B — Phonetic chain organization
- Layer C — Syllabic and rhyme structure

## Data
- Torah (Sefaria, niqqud, no taamim)
- Modern Hebrew control (HaGamad)

## Structure
- `src/` — analysis code
- `run/` — execution scripts (.bat)
- `data/` — input corpora
- `results/` — outputs
- `protocols/` — exact methods
- `docs/` — figures / paper materials

## Reproducibility
All results can be reproduced by running scripts in `run/`.

## Status
Baseline analyses completed (Layers A–C).  
Further validation and publication in progress.
